# 智能式三自由度机器人手腕设计

## Intelligent Design of a 3-DOF Robot Wrist

本科毕业设计项目 —— 智能式三自由度机器人手腕的结构设计与分析。

### 项目内容

- **三维模型**：装配体 STEP 文件
- **工程图纸**：装配图 A1、下层转动座 A3、弹性体 A3（DWG + PDF 格式）
- **有限元分析**：4×4 截面的静力学/瞬态/模态/谐波响应分析（ANSYS Workbench）
- **运动仿真**：机构运动动画
- **论文与答辩**：答辩 PPT

### 文件说明

| 文件 | 说明 |
|------|------|
| `装配体.STEP` | 三自由度手腕三维装配模型 |
| `装配图A1.dwg/.pdf` | 总装配工程图 |
| `下层转动座A3.dwg` | 下层转动座零件图 |
| `弹性体A3.dwg/.pdf` | 弹性体零件图 |
| `梁嘉楠毕业答辩：智能式三自由度机器人手腕设计.pptx` | 答辩演示文稿 |

| `4×4截面/` | ANSYS 有限元分析项目文件（不含大型结果文件） |
